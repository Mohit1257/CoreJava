# Java If-Else Practice Programs

This repository contains **basic Java programs** created to practice
**if-else conditions** and simple logic building.

I am currently learning **Core Java fundamentals**, so all programs are
written using **basic if-else statements**.

## Topics Covered

1. Check Number is Even  
2. Check Number is Odd  
3. Check Number is Even or Odd  
4. Check Number is Positive, Negative or Zero  
5. Find Smallest of Two Numbers  
6. Find Largest of Two Numbers  
7. Sum of Two Even Numbers  
8. Sum of Two Odd Numbers  
9. Sum of Even and Odd Numbers  
10. Check Salary Bonus  
11. Check Number is Single Digit or Double Digit  
12. Check Leap Year  
13. Find Greatest of Three Numbers  
14. Check Eligible for Voting  
15. Check Pass or Fail  

## Purpose

- Practice basic decision-making logic  
- Improve understanding of if-else conditions  
- Build strong fundamentals in Java  

This repository is created only for **learning and practice**.
